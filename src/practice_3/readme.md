1. create simple todo list app. 
2. todo item must have next structure: 

{
    title: "Visit all lectures", 
    createdAt: "15.02.2020", 
    dueDate: "16.02.2020",
    isDone: false
}

    3.1 add new todo list item;
    3.2 delete todo list item;
    3.3 mark todo list item as done.
